# HVAC AI Visibility MVP

Operator-first web app for running AI visibility audits for HVAC companies across:

- Gemini Maps Grounding
- Gemini Google Search Grounding
- ChatGPT Search
- Google AI Overview / SERP provider

The MVP intentionally skips automated website crawling. Operators manually enter the company profile, primary location, services, optional additional locations, and optional known competitors.
